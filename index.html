AI-Based Intrusion Detection System

Designed an AI-powered IDS analyzing network traffic to detect brute-force FTP/SSH attacks using the CSE-CIC-IDS2018 benchmark dataset. Engineered 79+ network flow features and evaluated ANN, Logistic Regression, and Random Forest models.

  • Engineered an end-to-end Python intrusion detection pipeline on the CSE-CIC-IDS2018 dataset, classifying FTP and SSH brute-force traffic with TensorFlow/Keras ANN, Random Forest, and Logistic Regression models
  • Built a 4-layer feed-forward Artificial Neural Network with dropout regularization and early stopping over 79 flow features using StandardScaler and an 80/20 stratified train-test split
  • Benchmarked three supervised classifiers on accuracy, precision, recall, and F1 — achieving ~99% accuracy with Random Forest and strong test-set performance across key evaluation metrics
